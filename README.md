# [爱与距离](https://weiinnar.github.io/)

爱情不是每一次都会有圆满的结局。真实的生活里，有人因为理想分开，有人因为性格错开。可正是这份不确定，让爱情更加令人珍惜。

相爱是一种勇气，也是一种学习。学会理解，而不是占有；学会尊重，而不是要求。两个人能在各自独立的人生中，依然选择靠近彼此——那才是爱情最成熟的模样。

有时候，爱需要距离。不是为了逃避，而是为了让彼此看清：原来，思念也可以是一种力量。若有一天重逢，那份感情或许更深，因为它曾经被时间沉淀，被孤独打磨。

爱情从不是命运的偶然，它是两颗心在一次次选择后的结果。即使结局不同，只要曾真心爱过，便已值得一生温柔回望。
